git status - показує стан файлів
git init - ініціалізує репозиторій
git add - додає вміст робочого репозиторія в індекс для наступного комміта
git commit - бере всі дані, добавлені в індекс командою "git add" і зберігає їх зліпок у вінутрішній базі даних, і потім зсуває вказівник вітки на цей зліпок
git branch - вміє перераховувати гілки, створювати нові, видаляти і перейменовувати їх
git push - використовується для публікації змін в репозиторії
git pull - отримує дані з сервера, з якого ви спочатку клонували, і автоматично намагається злити їх з кодом, над яким в даний момент працює користувач
