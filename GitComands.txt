git status - показує стан файлів
git init - ініціалізує репозиторій
git add - додає вміст робочого репозиторія в індекс для наступного комміта
git commit - бере всі дані, добавлені в індекс командою "git add" і зберігає їх
git branch - показує гілку на котрій ти зараз знаходишся
git push - використовується для публікації змін в локальний репозиторій
git pull - стягує зміни з локального репозиторія
git help - відображає список git-команд і їхнє значення
git merge <name_branch>- з'єднує вітки комітів в одну
git checkout <name_branch> - використовується для переключення між вітками
git checkout -b <name_branch> - використовується для створення і одразу вибору цієї вітки
git checkout -- <name_file> - відміна змін
git remote - дозволяє з'єднати локальний репозиторій з віддаленим
git log - показує історію комітів
